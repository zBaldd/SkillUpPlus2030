Tecnologias Utilizadas

- Google Gemini AI API
- Async Storage
- Expo Vector Icons


Como rodar o projeto localmente
comandos:
npm install -g expo-cli


Clonar o repositório
git clone https://github.com/SEU_USUARIO/SkillUpPlus2030.git


Entre na pasta:
cd SkillUpPlus2030


Instalar dependências
npm install


Rodar o projeto
npx expo start
